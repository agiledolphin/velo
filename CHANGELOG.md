# 变更日志

本文档记录 Velo / 微落的重要版本变化。版本号遵循[语义化版本](https://semver.org/lang/zh-CN/)，格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

## Unreleased

### 计划

- 接入 yt-dlp，实现真实媒体信息解析。
- 为外部进程、远程资源和内容安全策略建立明确边界。

## 0.1.0 - 2026-07-13

首个 Preview 版本，完成桌面应用骨架和模拟解析闭环。

### 新增

- 基于 Tauri 2、React 19、TypeScript 7 和 Rust 的桌面项目。
- Bun 依赖管理、开发、测试和生产构建脚本。
- Rust 领域模型、应用接口、Tauri 命令和 Mock 媒体引擎。
- URL 校验以及空闲、解析中、错误和成功状态。
- 媒体摘要、模拟格式列表与格式选择界面。
- 浏览器开发模式下的只读预览 fixture。
- 前端 URL 单元测试和 Rust 媒体引擎测试。
- 产品说明与第一阶段架构文档。

### 调整

- 完成 Velo / 微落首版品牌界面和 Slogan 排版。
- 使用自定义顶部区域替代传统标题栏，并支持窗口拖动。
- 将默认窗口尺寸设为 `1120 × 660`，最小尺寸设为 `920 × 600`。
- 将结果区域改为随解析状态变化的紧凑布局。
- 锁定中英文 Slogan 的左侧位置，避免随窗口宽度移动。

### 限制

- 当前使用模拟引擎，不发起真实网络解析或视频下载。
- 尚未接入 yt-dlp、FFmpeg、Cookie、下载历史和自动更新。
